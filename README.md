## 📌 프로젝트 소개
RE:Factory는 AI 기반 코드 구조 시각화 및 자동 리팩토링 플랫폼입니다.
소스 코드를 입력하면 정적 분석 → 구조 시각화 → 품질 평가 → 리팩토링 제안까지
5단계 파이프라인을 통해 코드 품질을 자동으로 분석합니다.

---

## 🛠 기술 스택

| 영역 | 기술 |
|---|---|
| 코드 분석 | Tree-sitter, Python AST |
| 구조 시각화 | Graphviz, NetworkX |
| 프론트엔드 | React, JavaScript |
| 백엔드 | Spring Boot |
| UI 설계 | Figma |
| 협업 관리 | Notion, GitHub |

---

## ⚙️ 핵심 기능

- **코드 구조 시각화** : 함수 호출 관계를 Fan-in/Fan-out 그래프로 표현
- **코드 품질 분석** : Cyclomatic Complexity, Code Smell 자동 탐지
- **리팩토링 제안** : 탐지된 문제 유형별 개선 코드 자동 제안
- **소셜 로그인** : GitHub, Google, Kakao, Naver 지원
- **다중 언어 지원** : Python, Java, JavaScript, TypeScript

---

## 📊 분석 파이프라인
코드 입력 → Tree-sitter 파싱 → AST 노드 탐색
→ 호출 관계 추출 → NetworkX 그래프 구성 → Graphviz 시각화 출력
